## Features

- **Main Application View**: This view shows the main StoreFlow Analytics workspace, including the analytics area and the system log section.

- **System Log View**: This view shows runtime messages, action results, errors, and status updates generated while the program is running.

- **Theme Selection View**: This view shows the available application themes: Dark, Light, Default, and Retro.

- **Add Store View**: This view shows the form used to create a new store by entering a store name and owner.

- **Camera Merge View**: This view shows the camera video merge setup, where users can import videos, choose which ones to merge, 
reposition them, preview the result, and save the merged video to a selected store.

- **Zone Setup View**: This view shows the aisle and zone setup interface, where users can load store zones, add or remove zones, move and 
resize them, preview them over video, and save them to the database.

- **Pie Chart View**: This view shows the selected filtered data as a pie chart based on the chosen categories, counts, and time range.

- **Heatmap View**: This view shows customer-density heatmaps for a selected store, background, and hour range.

- **Heatmap Video View**: This view shows a rendered heatmap video over time, with playback controls and summary information.

- **Customer Aisle Analytics View**: This view shows customer attribute summaries by aisle, including the most common gender and age group for each aisle.

- **Customer Product Analytics View**: This view shows customer attribute summaries by product, including the most common gender and age group associated with each product.

- **Customer Attributes Estimator View**: This view shows the label sets used by the customer attribute estimation module.

- **Section Time Analysis View**: This view shows how long customers spend in different sections of each aisle, including total time, average time, and customer count.

- **Basket Analysis Summary View**: This view shows overall basket metrics such as total transactions, total revenue, average basket size, average basket quantity, and average basket value.

- **Basket Analysis Per Product View**: This view shows basket-related sales data for each product, including quantities sold, transactions, and revenue.

- **Basket Analysis Product Pairs View**: This view shows which products are commonly bought together using co-occurrence, support, confidence, and lift metrics.

- **Revenue Summary View**: This view shows high-level revenue metrics such as total revenue, transaction count, average transaction value, top product, and peak revenue period.

- **Revenue by Time View**: This view shows revenue over time using line charts, bar charts, and data tables with selectable time granularity and window ranges.

- **Revenue by Product View**: This view shows revenue breakdown by product, along with product filters and chart/table summaries.

- **Revenue by Customer Age View**: This view shows how revenue is distributed across customer age groups.

- **Revenue by Customer Sex View**: This view shows how revenue is distributed across customer sex categories.

- **Simulation View**: This view shows simulation output, including render controls, simulation details, aisle/category information, and video playback controls.

- **Database Management View**: This view shows options for maintaining the local database, including database wipe functionality and CSV-based sales-data import.

- **Help / Wiki View**: This view shows access to the project Wiki/Readme and maintenance tools such as orphaned video cleanup.
